Detailed Description

template<typename T>
class PolyWrapper< T >

A wrapper to hold pointers to heap-allocated objects on the stack.

A wrapper to hold a pointer to a heap allocated object. When this object is copied, so is the heap allocated object it points to.

Parameters:
TA base class that the heap-allocated objects should derive from
